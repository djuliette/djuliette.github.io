The point of this track is to run a pre-trained model and see how it performs.
Pressing the l key (lower case l) will load a model trained on the original track.
After loading you can see that the agents have no difficutly completing this track
even though they have never seen this environment before.

So First if you hit the 1 it will stop drawing all sensors so it can run faster.
You can also press the c key to stop/start the camera tracking so you can explore the track.
Now you can see the agents going around the track.

If you are patient enough they will complete the track usually around 50 generations.  They can solve the environment sooner if by random mutation they get lucky.
